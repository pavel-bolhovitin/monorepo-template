# Common Rules

Apply these rules unless the user explicitly requests otherwise.

## Language

- **Rules**: Write all rule content in **English**.
- **Code**: Write all code (identifiers, strings in code, test names) in **English**.
- **Comments**: Write all comments in **English**.
- **Documentation**: Write all documentation (README, docs, JSDoc) in **English**.

Use another language only when the user explicitly asks for it (e.g. UI copy, localized content, or rule language).

## Files under `.agents/`

- **Documentation Map:** When you add any new file under `.agents/`, you MUST add it to the Documentation Map in [.agents/agents.md](./agents.md) so agents can discover and use it.
